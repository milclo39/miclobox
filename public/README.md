# AR Drawing Web App

ブラウザ上で動作する軽量なAR（拡張現実）ドローイングアプリケーションです。
デバイスのカメラ映像やアップロードした画像の上に、手書きのイラストやメモを描くことができます。
ズーム・パン機能（無限キャンバス風）や、録画・静止画保存機能を備えています。

## 📸 機能概要

### 1. ドローイング機能

* **ペンツール**: 8色のプリセットカラー、4段階の太さ、4段階の不透明度を選択可能。
* **消しゴム**: 描いた線を部分的に消去。
* **Undo**: 直前の操作を取り消し。
* **全消去**: キャンバスをリセット。
* **自動保存**: 描画内容は `LocalStorage` に保存され、リロードしても復元されます。

### 2. 背景モード (AR / 画像)

* **カメラモード (AR)**: デバイスのカメラ映像を背景としてリアルタイム表示。
* **カメラ切り替え**: 複数のカメラ（背面・前面・広角など）を検出して切り替え可能。
* **解像度設定**: 640x480 から 4K (3840x2160) まで、カメラの解像度をリクエスト可能（デバイスの対応状況による）。
* **実解像度表示**: 実際に適用された解像度を表示。


* **画像モード**: ローカルの画像をアップロードして下絵として使用可能。
* **ホワイトボードモード**: 背景なし（白）の状態で描画。

### 3. スマートズーム & パン

* **ピンチイン・アウト**: 2本指の中心点を基準に拡大縮小（スマホ操作ライクな直感的な挙動）。
* **パン（移動）**: ズーム中は1本指（またはマウスドラッグ）でキャンバスを自由に移動可能。
* **スライダー操作**: 画面下部のスライダーで画面中央を基準に拡大縮小。

### 4. 保存・エクスポート

* **スナップショット**: 背景と描画を合成した静止画 (PNG) をダウンロード。
* **動画録画**: 描画の様子を背景ごと録画し、動画ファイル (WebM) としてダウンロード。

## 🛠 技術スタック

* **言語**: HTML5, CSS3, JavaScript (Vanilla JS - ライブラリ不使用)
* **API**:
* `Canvas API`: 描画および画像合成処理。
* `MediaDevices API (getUserMedia)`: カメラ映像の取得。
* `Pointer Events API`: タッチ・マウス・ペン入力の統一処理。
* `MediaRecorder API`: キャンバスの動画キャプチャ。



## 📖 使い方

### メインコントロール

| アイコン | 機能 | 説明 |
| --- | --- | --- |
| 🔍 | ズームモード切替 | 描画モードと移動/拡大モードを切り替えます。 |
| 🖼️ | 背景設定 | カメラの選択、画像のアップロード、解像度設定を行います。 |
| ✏️ | ペン | 描画モードになり、ペンメニューを開きます。 |
| 🧼 | 消しゴム | 消しゴムモードにします。 |
| ↩️ | Undo | 一つ前の操作に戻ります。 |
| 🗑️ | 全消去 | 全ての描画をクリアします。 |

### 操作ジェスチャー

* **1本指ドラッグ**:
* ズームモードOFF時：線の描画。
* ズームモードON時：キャンバスの移動（パン）。


* **2本指ピンチ**:
* ズームモードON時：拡大・縮小（ピンチの中心に向かってズームします）。



### 解像度の変更方法

1. `🖼️` ボタンを押して背景メニューを開く。
2. カメラを選択して起動する。
3. メニュー下部の「解像度設定」ボタン（例: `1920x1080`）を押す。
4. カメラが再起動し、指定された解像度（`ideal`値）での接続を試みます。

## ⚙️ セットアップ & 実行方法

本アプリは単一のHTMLファイル (`index.html`) で構成されています。

1. リポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開きます。

### ⚠️ 注意事項: カメラ動作環境について

セキュリティ上の制約により、ブラウザからカメラにアクセスするには以下のいずれかの条件が必要です。

* `https://` で配信されていること（GitHub Pages, Vercel, Netlify等）。
* `localhost` (ローカル開発環境) であること。

※ ローカルのHTMLファイルを直接ダブルクリック (`file://...`) して開いた場合、カメラ機能は動作しません。VS Codeの "Live Server" 拡張機能などを利用してローカルサーバー経由で開いてください。

## 🧩 内部アーキテクチャ

### レイヤー構造

`#viewport` 内に以下の順序で配置されています。

1. **Zoom Container**: `transform` プロパティで拡大・移動を制御するラッパー。
2. **Background Canvas (`z-index: 0`)**: カメラ映像や画像を描画。
3. **Drawing Canvas (`z-index: 1`)**: ユーザーの描画パスを表示。

### ズームロジック

拡大縮小時は、常に「指定された中心点（画面中央またはピンチの中点）」が、拡大後も同じスクリーン座標に留まるように `translate` 値を逆算して補正しています。

```javascript
// 概念コード
const contentX = (center.x - translateX) / oldScale;
const contentY = (center.y - translateY) / oldScale;
// 新しい移動量を計算
newTranslateX = center.x - (contentX * newScale);
newTranslateY = center.y - (contentY * newScale);

```
